# Humanizer RU

Skill для Claude Code, который удаляет признаки AI-генерации из русского текста, делая его живым и человечным.

## Версия 1.2 - зачем обновили

Модели меняются. Каждое новое поколение пишет код всё лучше, а естественным языком - всё хуже: их затачивают под программирование, и эта заточка протекает в обычную речь. Модель объясняет отпуск как pull request: «скорость > красоты», «идея → прототип → продукт», «итерируемся по гипотезам». Человек так не говорит. Версия 1.2 бьёт именно по этому.

### Добавлено 9 паттернов (22-30)

- **Стоп-слова и усилители** - «в современном мире», «действительно», «безусловно». Удалил слово, смысл не изменился - слово было мусором
- **Математические и кодовые знаки [жёсткий запрет]** - равенства, стрелки, знаки сравнения, «vs» и плюсы в обычном тексте. Всё пишется словами, есть таблица замен
- **Техножаргон без пояснения** - «валидируем», «дефолтный», «флоу». Правило: пиши для умного 12-летнего подростка. Он поймёт любую мысль, если сказать по-человечески. Научный термин можно, но при первом появлении поясни одной фразой
- **Псевдоглубина** - «по сути», «настоящий вопрос в том». Торжественная подводка, за которой прячется обычная мысль
- **Анонсы вместо дела** - «давайте разберёмся», «вот что нужно знать». Модель объявляет, что сейчас сделает, вместо того чтобы делать
- **Рубленый драматизм** - стопки коротких фрагментов подряд: «Без правил. Без границ. Только результат». Одно короткое предложение для акцента работает, три подряд - спектакль
- **Формулы афоризмов** - «внимание - валюта XXI века». Цитата для мотивационного постера вместо конкретного утверждения
- **Фальшивая доверительность** - «Честно?», «Скажу прямо». Театральная пауза перед обычной мыслью
- **Заголовок и его пересказ** - строка после заголовка, которая пересказывает заголовок

### Изменено

- **«Оценка меняется на факт»** в рекламном языке: «прочный» превращается в «выдерживает 150 кг». Свой вывод читатель запомнит, чужой - нет (из «Пиши, сокращай» Ильяхова)
- **«Одно предложение - одна мысль. Один абзац - одна тема»** в требованиях к стилю
- **Проверка на уровне абзацев**: главная мысль в начале; абзац не даёт читателю ничего нового - удаляется целиком
- **Раздел «Что не трогать»** - защита от переусердствования. Искать кластеры признаков, а не одиночные слова. Цитаты, странные конкретные детали, смешанные чувства и самопоправки - признаки живого человека, их сохраняем
- **Шаг самопроверки в процессе**: перед выдачей спросить себя «что в этом тексте всё ещё выдаёт AI» и переписать найденное
- **«Режь воду, а не фактуру»**: факты, цифры и примеры оригинала остаются

## Установка

### Рекомендуемый способ

```bash
git clone https://github.com/smixs/humanizer-ru.git ~/.claude/skills/humanizer-ru
```

### Ручная установка

```bash
mkdir -p ~/.claude/skills/humanizer-ru
curl -o ~/.claude/skills/humanizer-ru/SKILL.md https://raw.githubusercontent.com/smixs/humanizer-ru/main/SKILL.md
```


## 30 паттернов AI-текста

### Канцелярит
1. Отглагольные существительные вместо глаголов
2. Цепочки родительных падежей
3. Избегание "это"/"есть" (Copula Avoidance)
4. Пассивные конструкции вместо активных
5. Иностранные слова там, где есть русские

### AI-словарь и раздувание
6. Перегруженные слова (AI Vocabulary)
7. Раздувание значимости и известности
8. Поверхностный анализ с -ание/-ение/-ая
9. Рекламный язык
10. Размытые атрибуции (Weasel Words)
11. Шаблонные переходы
12. Формульные выводы о "вызовах и перспективах"

### Структурные
13. Негативные параллелизмы [ЖЁСТКИЙ ЗАПРЕТ]
14. Правило трёх
15. Элегантная вариация (Synonym Cycling)
16. Ложные диапазоны

### Тон и коммуникация
17. Подобострастный тон (Sycophancy)
18. Артефакты чатбота
19. Хеджирование (перестраховка)
20. Общие позитивные заключения
21. Эмодзи и избыточное форматирование

### Новое в 1.2
22. Стоп-слова и усилители
23. Математические и кодовые знаки [ЖЁСТКИЙ ЗАПРЕТ]
24. Техножаргон без пояснения
25. Псевдоглубина
26. Анонсы вместо дела
27. Рубленый драматизм
28. Формулы афоризмов
29. Фальшивая доверительность
30. Заголовок и его пересказ

## Жёсткие запреты

- "не просто X, а Y"
- "не только X, но и Y"
- длинное тире "—" (только короткое "-")
- риторические вопросы
- математические и кодовые знаки в тексте: = → > < + vs &
- техножаргон без пояснения по-человечески
- стопки рубленых фрагментов подряд («Без X. Без Y. Только Z.»)
